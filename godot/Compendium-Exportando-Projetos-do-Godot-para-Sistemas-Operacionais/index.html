Compendium - Exportando Projetos do Godot para Sistemas Operacionais

Nesse documento você vai aprender a como exportar seu projeto do Godot para as plataformas Windows, Mac e Linux.

Depois de terminar seu projeto, pode ser necessário que o exporte para rodar nos principais sistemas operacionais de computadores. Os sistemas que o Godot consegue exportar como padrão são Windows,Mac e Linux.

Para começar verifique se o seu projeto possui uma cena principal selecionada para abrir automaticamente quando o jogo iniciar. Você pode fazer isso indo na guia File System e com o botão direito do mouse em qualquer cena selecionar a opção Set as Main Scene.

Depois de ter selecionado sua cena principal podemos exportar o projeto. Basta ir no menu Project > Export:

Dentro da janela que será mostrada você verá apenas um vazio. Para exportar é necessário antes escolher a plataforma, então basta ir em Add para escolher:

Observação: o exportador correto para windows é o Windows Desktop.

Se por acaso aparecer alguns avisos em vermelho na parte superior, pode ser que você não tenha instalado o pacote de templates de exportação do Godot. Para resolver é simples: Primeiro você deve clicar em Manage Export Templates.

Depois basta selecionar a versão que deve ser feito o download do pacote, e clicar no link. Dessa forma o próprio Godot vai fazer o download dos arquivos necessários:

Depois de selecionado a plataforma, todas terão opções parecidas para marcar. Uma delas é o Runnable. Deixe ela marcada. Depois é necessário que você escolha o local onde serão salvos os arquivos da exportação. É recomendado que seja criado antes uma pasta na Área de Trabalho com o nome da plataforma para que dentro dela seja salvo todo os arquivos:

Criado a pasta, selecione o caminho onde será salvo os arquivos em Export Path. Assim que escolher a pasta insira também o nome do projeto:

Na parte esquerda dentro das plataformas Windows e Mac existe a caixa Application, onde você pode ainda inserir mais informações do seu projeto, como descrição, nome do desenvolvedor, trocar o ícone, etc. Mas não é de preenchimento obrigatório.

Depois de escolher a pasta de destino e por o nome do projeto, basta exportar. Clique em Export Project na parte inferior e o Godot irá exportar o jogo dentro da plataforma selecionada.

Na janela que aparecer confira o nome do arquivo e a pasta que será salvo. Apenas desmarque a opção Export with Debug e clique em salvar:
